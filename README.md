# 🧰 Ultimate Web Development Toolkit (HTML, CSS, JavaScript)

Una colección curada de herramientas esenciales para desarrollo web moderno: frontend, backend, testing, rendimiento y productividad.

---

## 🚀 Terminal Tools (CLI)

### 📦 Package Managers

* npm
  Gestor de paquetes por defecto de Node.js. Permite instalar, actualizar y manejar dependencias en proyectos JavaScript.

* Yarn
  Alternativa a npm con mejor manejo de caché y velocidad en instalaciones.

* pnpm
  Optimiza el uso de disco reutilizando dependencias mediante enlaces simbólicos.

---

### 🔍 Linting & Formatting

* ESLint
  Analiza el código JavaScript para detectar errores, malas prácticas y mantener estándares.

* Prettier
  Formatea automáticamente el código para mantener consistencia visual sin discusión.

---

### 🧪 Testing

* Jest
  Framework de testing popular para JavaScript, ideal para pruebas unitarias.

* Vitest
  Alternativa moderna y rápida a Jest, integrada con Vite.

* Playwright
  Permite pruebas end-to-end automatizando navegadores reales.

---

### ⚙️ Build Tools

* Vite
  Herramienta de desarrollo rápida con hot reload casi instantáneo.

* Webpack
  Empaquetador altamente configurable para proyectos complejos.

* Parcel
  Bundler simple que funciona sin configuración inicial.

---

### 🌐 HTTP / Debug

* curl
  Herramienta de línea de comandos para hacer peticiones HTTP y probar APIs.

* HTTPie
  Cliente HTTP más legible que curl, pensado para productividad.

---

## 🌍 Web Tools (Online)

### 🎨 CSS & UI

* CSS Gradient
  Generador visual de gradientes CSS listo para copiar.

* Clippy
  Permite crear formas complejas con `clip-path` fácilmente.

* Glassmorphism
  Generador de efectos modernos tipo vidrio (blur + transparencia).

---

### 🎯 Layout & Flex/Grid

* Flexbox Froggy
  Juego interactivo para aprender Flexbox de forma práctica.

* CSS Grid Generator
  Herramienta visual para crear layouts con CSS Grid.

---

### 🔤 Fonts & Icons

* Google Fonts
  Biblioteca gratuita de tipografías optimizadas para web.

* Font Awesome
  Colección masiva de iconos listos para usar.

* Heroicons
  Iconos SVG modernos y minimalistas.

---

### 🧪 Testing & Debugging

* Can I Use
  Muestra compatibilidad de tecnologías web entre navegadores.

* Lighthouse
  Analiza rendimiento, accesibilidad y SEO de una web.

* WebPageTest
  Herramienta avanzada para medir tiempos de carga reales.

---

## 🧠 HTML Tools

* HTML Validator
  Verifica que tu HTML cumpla con los estándares oficiales.

* Emmet
  Permite escribir HTML y CSS rápidamente con abreviaciones.

* CodePen
  Editor online para prototipos rápidos y pruebas visuales.

---

## 🎨 CSS Tools

* Sass
  Añade variables, funciones y lógica al CSS tradicional.

* PostCSS
  Transforma CSS con plugins (autoprefixer, optimización, etc.).

* Tailwind CSS
  Framework basado en utilidades para diseñar rápido sin escribir CSS personalizado.

---

## ⚡ JavaScript Tools

* Node.js
  Permite ejecutar JavaScript fuera del navegador.

* TypeScript
  Añade tipado estático para evitar errores en proyectos grandes.

* Axios
  Cliente HTTP basado en promesas para consumir APIs fácilmente.

---

## 🧱 Frameworks & Libraries

### Frontend

* React
  Librería para construir interfaces interactivas basadas en componentes.

* Vue.js
  Framework progresivo fácil de aprender para interfaces modernas.

* Angular
  Framework completo mantenido por Google para aplicaciones grandes.

---

### Backend

* Express.js
  Framework minimalista para crear APIs y servidores con Node.js.

* Next.js
  Framework para React con SSR, routing y optimización integrada.

---

## 🧰 Dev Utilities

* Git
  Sistema de control de versiones para gestionar cambios en el código.

* Docker
  Permite ejecutar aplicaciones en contenedores aislados.

* Visual Studio Code
  Editor de código extensible y muy usado en desarrollo web.

---

## 🔥 Bonus (Productividad)

* Figma
  Herramienta colaborativa para diseño de interfaces.

* Notion
  Plataforma todo-en-uno para notas, documentación y organización.

* Obsidian
  Sistema de notas basado en markdown con enlaces tipo grafo.

---

## 🧠 Filosofía básica

* HTML → estructura
* CSS → presentación
* JavaScript → comportamiento
